# Saudi Tourism App 🇸🇦 ✨
**A Premium Flutter Application for Exploring Saudi Arabia's Treasures**

This project was developed as part of a high-performance bootcamp, demonstrating advanced skills in **Flutter UI/UX design**, **Clean Architecture**, and **Backend Integration** with Supabase.

---

## 🏆 Key Achievements
To ensure the highest quality, this project adheres to the following standards:

### 1. 🎨 Professional UI/UX
* **Consistent Brand Identity**: Custom color palette (`AppColors`) inspired by the modern Saudi identity (Royal Green & Desert Sand).
* **User Experience**: Smooth transitions, intuitive navigation, and responsive layouts.

### 2. 🏗️ Clean Architecture & Code Quality
* **Folder Structure**: Logically separated into `models/`, `services/`, `screens/`, `widgets/`, and `constants/` .
* **Reusable Components**: Developed custom, high-quality widgets like `CustomButton` to maintain code DRY (Don't Repeat Yourself).
* **Type Safety**: All data is parsed using strong Models instead of raw Maps.

### 3. ⚡ Technical Integration (Supabase)
* **Authentication**: Fully functional Sign-Up and Login system with session persistence .
* **Real-time Database**: Dynamic data fetching from Supabase with proper error handling and loading states.
